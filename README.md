# Практическая работа №6 - Системная утилита на Bash

Утилита для отображения информации о пользователях и процессах системы.

## Функциональность

- Просмотр списка пользователей и их домашних директорий
- Просмотр запущенных процессов с сортировкой по PID
- Сохранение вывода в файл
- Сохранение ошибок в отдельный файл

## Использование

```bash
chmod +x main.sh
./main.sh -h
